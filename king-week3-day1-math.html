Week 3 Day 1: Equivalent Fractions

"What's up, striker! Sometimes a play looks different but equals the same result. Fractions are the same way! This is called equivalent fractions."
PHASE 1: Same Value

Equivalent fractions are fractions that look different but have the exact same value.

üîë KEY POINT: 1/2 is the same as 2/4 and 4/8.
PHASE 2: Making Equivalent Fractions

Multiply or divide the top and bottom numbers by the same number.

‚öΩ MATH TIP: If you do it to the top, do it to the bottom!
PHASE 3: Soccer Example

If you need 4/8 of a game to feel warmed up, that is the same as needing 1/2 of the game!

PHASE 4: Practice Problems

1. Find a fraction equivalent to 1/3 (Multiply by 2).

2. Simplify 6/10 (Divide by 2).

PHASE 5: Physical Mission

Set up 10 cones. Run to 5 of them. Write that fraction (5/10). Then, find the equivalent simplified fraction (1/2).
